# SSWrapper
SSKit中用于基础结构搭建的自定义组件，各个组件都高度灵活可自定义，目前正在不断完善中与开发新的控件

**<u>组件结构：</u>**

- SSAlertView
- SSDialog
- SSPageControl
- SSProgressIndicator
- SSPullRefresh
- SSStatus
- SSTabbar
- SSInherits
- SSBugCollector - 未完成

**<u>使用：</u>**

```
pod 'SSWrapper'
```
